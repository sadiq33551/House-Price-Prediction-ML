# House Price Prediction

## Objective
Predict house prices using property features such as size (in sqft) and number of bedrooms.  
The project demonstrates a simple regression model using Python and scikit-learn.

## Dataset
- A small sample dataset is used in the script.
- Columns:
  - `Size_sqft` → Area of the house in square feet
  - `Bedrooms` → Number of bedrooms
  - `Price` → Target variable (House price in USD)

## Model Used
- Linear Regression (from scikit-learn)

## Evaluation
- Mean Absolute Error (MAE) is calculated to check prediction accuracy.

## How to Run
1. Open `house_price.py` in Python (VS Code, Jupyter Notebook, or terminal)
2. Run the script
3. Predicted prices, actual prices, and MAE will be displayed in the console

## Technologies Used
- Python
- Pandas
- Scikit-learn
